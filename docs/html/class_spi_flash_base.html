Detailed Description

Pure virtual base class SPI for SpiFlash devices.

Library for SPI NOR flash chips for the Particle platform

https://github.com/rickkas7/SpiFlashRK

License: MIT This is used because the P1 flash uses the system firmware flash API instead of the low-level API in the SpiFlash class, so this provides a common API.

Erases the entire chip.

This call blocks for the duration of the erase, which take take some time (several secoonds). This function uses delay(1) so the cloud connection will be serviced in non-system-threaded mode.
